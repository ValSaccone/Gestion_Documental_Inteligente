Subproyecto titulado “Módulo de Gestión Documental Inteligente para la Automatización de Registros de Facturas”. El mismo se encuentra dentro del proyecto “Agentes Inteligentes como Servicio (AIAAS) para PyMEs” perteneciente al grupo de GIITNI en UTN-FRT. El subproyecto busca implementar un módulo de gestión documental de comprobantes fiscales con foco en facturas, para automatizar la carga manual de registros mediante técnicas de reconocimiento óptico de caracteres (OCR). Esto incluye procesamiento de imágenes de facturas, detección de regiones de interés, extracción de datos clave y almacenamiento de esta información en una base de datos relacional.
